## 物体間の関係性を考慮した車載カメラ画像からのリスク要因推定

### フォルダやファイルの説明

#### paper フォルダ

論文執筆関連のファイルを収録

- texファイル(main.tex)
  - 論文の本文をtex言語で記述したテキストファイル
    
- styファイル(mthesis.tex)
  - texファイルから出力される文書のスタイルやレイアウトの設定を記述したファイル
  - main.texで呼び出される
    
- bibファイル(refs.bib)
  - 参考文献を一括管理するためのファイル
  - main.texで呼び出される
